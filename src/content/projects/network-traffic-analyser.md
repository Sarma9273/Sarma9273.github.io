## Goal

Convert raw traffic-related information into clear answers: what happened, which systems were involved, how serious it is and what should be investigated next.

## Delivered components

- Traffic and alert summary cards.
- Normal versus suspicious activity views.
- Protocol and source-address statistics.
- Severity classification and recent-incident tables.
- Analyst notes and recommended next actions.

## Learning value

The project connected Python data processing with SOC workflow design. It also required debugging large notebook code blocks, incomplete multiline strings and mobile-responsive presentation.

## Roadmap

PCAP ingestion, live capture, scan and brute-force detection, IP reputation enrichment, Zeek or Suricata integration, exportable reports and anomaly detection.
