## Classroom problem

Copying exercises and examples separately to every lab computer wastes teaching time and creates inconsistent file versions.

## Solution

A shared network location presents organised folders for computer fundamentals, Python, HTML, CSS, assignments, media and teacher resources. Read-only areas and separate submission locations balance access with control.

## Operational considerations

The implementation accounts for permissions, backups, accidental deletion, stored credentials, inconsistent drive mappings and student-friendly naming.

## Roadmap

Individual student folders, automated backups, submission tracking, a local learning portal, browser coding exercises and lab-device health monitoring.
